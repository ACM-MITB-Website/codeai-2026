## üöÄ Getting Started

Follow these instructions to set up the project locally.

### Prerequisites
*   Node.js (v16 or higher)
*   npm (v7 or higher)

### Installation
1.  **Clone the repository**
    ```bash
    git clone https://github.com/ACM-MITB-Website/codeai-2026.git
    cd codeai-2026
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

### Development
Start the local development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### Production Build
To create an optimized build for production:
```bash
npm run build
```
The output will be generated in the `dist` directory (note: `dist` is git-ignored).
